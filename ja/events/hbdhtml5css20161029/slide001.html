Webの標準 ＝ 電子出版の標準へ

  • ２つの標準化団体の連携によりWeb標準を使う電子出版が実現
    • W3C (World Wide Web Consortium)
    • IDPF (International Digital Publishing Forum)
  • Webと出版が融合？
    • 「Web出版物」の標準化構想
      Portable Web Publications for the Open Web Platform
