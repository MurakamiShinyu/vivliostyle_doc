Vivliostyle 製品イメージ

Vivliostyle 製品計画

  • Vivliostyle.js CSS組版エンジン・コア
    • オープンソース Apache ライセンス
  • Vivliostyle.js ビューアUIエンジン、および全部込み（組版エンジン＋UI＋etc.）
    • GPLライセンス、または商用ライセンス（企業向けカスタマイズ、価格は応相談）
  • Vivliostyle Browser （CSS組版拡張ブラウザ）
    • GPLライセンス、または商用ライセンス（企業向けカスタマイズ、価格は応相談）
  • Vivliostyle Formatter（CSS自動組版PDF生成エンジン）
    • 商用ライセンス販売、サーバー版およびスタンドアロン版
  • Vivliostyle クラウド サービス（組版エンジンの機能をクラウドサービスで提供）
    • PDF出力ページ数ごとの課金など

製品開発状況

  • Vivliostyle.jsおよびVivliostyle Formatterのベータ評価版を2015年9月16日から公開中
  • 2015年12月製品リリース予定
    • Vivliostyle.js CSS組版エンジン、ビューアUI、etc.
    • Vivliostyle Formatter
  • 2016年春以降、次の製品・サービスをリリース予定
    • Vivliostyle Browser
    • Vivliostyle クラウド サービス

利用分野

  • 電子書籍+印刷書籍
  • Webコンテンツ＋オンデマンド印刷（カタログ、パンフ、会誌、タウン誌）
  • 電子雑誌・電子新聞・電子ブックビュワー
  • マニュアル、ドキュメント制作(多言語の大規模なドキュメントを大量に組版)
  • 学術論文
  • デジタル教科書・教材
  • Web-to-Printシステムに組み込む組版エンジンとして
  • その他、いろいろ
