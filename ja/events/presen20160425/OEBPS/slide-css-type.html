XML自動組版から
HTML5+CSS組版への発展

  • これまでのXML自動組版は、
    • DTPソフトのXML機能や専用のXML組版ソフトを利用。
    • 組版カスタマイズにはXML組版言語の知識が必要。
    • 紙・PDF用の組版（専用のXML）と、Web・EPUB電子出版の制作（HTML+CSS）をひとつにするのが難しい。
  • 今後の標準は、HTML5+CSS組版へ

CSS組版とは

  • WebページもEPUB電子書籍もHTML+CSSで出来ている。
  • 紙の本もHTML+CSSで組版して作ることが出来る。
    Vivliostyle以前にも印刷出版物を作れるHTML+CSS組版エンジンはあった。
  • Vivliostyle の新しいところは、標準Webブラウザベースで本格的な組版を可能にすること

Vivliostyle の技術

  • 次世代CSS組版
    • Webブラウザ上で高度な組版機能をサポートする JavaScript ライブラリ Vivliostyle.js
    • 次世代のCSS組版仕様の標準を、世界に先駆けて実装
  • Webと電子出版の標準仕様の技術の推進
    • Web標準技術が、世界の電子出版と組版仕様の基盤に
